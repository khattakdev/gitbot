> A CLI Based Application to help you learn Git.

## Prerequisite
You'll need to have [Nodejs](https://nodejs.org/) installed on your system.
## Install

```sh
npm install -g @khattakdev/gitbot
```

## Usage

Open an empty folder in your terminal, type:

```sh
gitbot
```
